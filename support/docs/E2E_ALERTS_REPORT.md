# E2E Alerts Report

This file was sanitized to avoid storing machine-specific values in Git history.

Removed from the tracked version:
- real IP addresses
- real usernames
- real passwords
- absolute home-directory paths
- environment-specific command transcripts

## Guidance

If you need a local end-to-end alerts report on a specific machine, generate it locally and store it outside tracked files, for example:
- `tmp/e2e_alerts_report.md`

Use environment variables instead of hardcoded host credentials when reproducing commands.

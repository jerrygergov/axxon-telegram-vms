Note

[Configuring the Web-Server](/confluence/spaces/one20en/pages/246484274/Configuring+the+Web-Server).

The default Web-Server port is **80** (Windows), **8000** (Lunix), the prefix is **/** (empty).

The request body should contain the gRPC API method (see [gRPC API](/confluence/spaces/one20en/pages/246487060/gRPC+API)).

The gRPC API can also be proxied via the Web-Server. The requests are described in the proto-files (see [gRPC API](/confluence/spaces/one20en/pages/246487060/gRPC+API)).
